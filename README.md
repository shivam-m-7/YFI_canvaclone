# YFI_canvaclone


# client - canva.com Homepage replica
# server - Library Management System

## Frontend Deploy
- https://canva-repl.netlify.app/


## API EndPoint
- POST - "/book/add"
    - Add
- GET - "/book/"
    -  get all the book list
- GET - "/book/:id" 
    - get book with given id
- PUT - "/book/:id" 
    -  update book with given id
- DELETE - "/book/:id"
    -  delete book with id
